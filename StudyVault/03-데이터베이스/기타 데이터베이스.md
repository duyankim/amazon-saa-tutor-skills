# 기타 데이터베이스 (★★)

#aws-saa #database

## Overview Table

| 서비스 | 유형 | 주요 특징 |
|--------|------|---------|
| **Redshift** | 열 기반 OLAP | 페타바이트 분석, RA3 노드 |
| **Neptune** | 그래프 | 소셜/추천/사기 감지 |
| **DocumentDB** | 문서(MongoDB 호환) | MongoDB 마이그레이션 |
| **Keyspaces** | 와이드 컬럼(Cassandra 호환) | Cassandra 마이그레이션 |
| **Timestream** | 시계열 | IoT/모니터링 |
| **QLDB** | 원장(불변) | 금융 거래 감사 |

---

## Redshift 핵심

```
Redshift Cluster:
  Leader Node (쿼리 계획/조정)
  Compute Nodes × N (실제 처리)

Redshift Spectrum:
  Redshift → S3 데이터 직접 쿼리 (별도 클러스터 불필요)
```

- 열 기반 압축 → 분석 쿼리 최적
- JDBC/ODBC 지원 → BI 도구 연동
- Aqua (Advanced Query Accelerator): 캐시 가속

---

## Neptune (그래프 DB)

- **SPARQL** (RDF) / **Gremlin** (Property Graph) 지원
- 사용 케이스: 소셜 네트워크, 추천 엔진, 사기 탐지, 지식 그래프
