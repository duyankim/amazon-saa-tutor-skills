### 오답 메모

**EC2 가격 모델 (Spot vs Reserved)**
- 혼동 (1차): 비용 최소화 + 중단 허용 → 예약 인스턴스를 선택
- 혼동 (2차): 주말 전용 배치 작업에 Reserved 선택 (간헐적 사용 = On-Demand가 유리)
- 핵심: 중단 허용 → Spot (최대 90% 절감) / 장기·안정적 워크로드 → Reserved / 간헐적·단기 → On-Demand
